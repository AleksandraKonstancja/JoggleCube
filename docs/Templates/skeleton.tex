\section{INTRODUCTION}
This document, produced using \LaTeXe document describes the use of
{\tt project.cls} which implements the document standard specified in
\cite{se.qa.03}.
\subsection{Availability}
The class file may be downloaded from the module web site on Blackboard.

\section{ADDITIONAL COMMANDS}
A number of additional commands to provide the standard information
required in the documents are provided. A typical header for a source
file is shown in Figure ~\ref{fig:header}. This will provide the front
material and running headers. To produce the \emph{Page n of m}
footer, the label {\tt thelastpage} must be inserted at the end of the
document. Figure ~\ref{fig:footer} shows the typical end of a
document. The un-numbered bibliography and document history sections
must be explicitly added to the table of contents.
